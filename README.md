# 🧠 Fraud Detection Model (Logistic Regression)

This is a mini project using Python and scikit-learn to detect credit card frauds.

## 🚀 Tech Stack
- Python
- scikit-learn
- pandas, numpy
- Logistic Regression

## 📊 Results
Confusion Matrix:
[[85279, 16], [59, 89]]

Precision (Fraud): 85%  
Recall (Fraud): 60%

## 📁 Files
- `fraud_model.py` → Main script
- `creditcard.csv` → Dataset (NOT included due to size)
